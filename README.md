# Zaiqa House — Restaurant Website

![Zaiqa House homepage](screenshot.png)

Dark-theme single-page site for a fictional Pakistani restaurant:
tabbed menu, gallery grid, story section, and a reservation form with
date/time/guest selection. Editorial serif + sans type pairing,
responsive throughout.

**Stack:** HTML5, Tailwind CSS (CDN), vanilla JavaScript.
